# AGENTS: Обязательные правила старта

Этот файл является обязательной точкой входа для любого AI-агента в репозитории.
Нельзя приступать к выполнению задачи, пока не пройден preflight-чеклист.

## Обязательный порядок чтения (до начала любой работы)
1. Прочитать [ai_agents/mandatory_instructions.md](/Users/mvkutepov/work/praktikum3/ai_agents/mandatory_instructions.md).
2. Пройти [ai_agents/preflight_checklist.md](/Users/mvkutepov/work/praktikum3/ai_agents/preflight_checklist.md).

## Правило жесткой остановки
Если preflight не завершен, агент обязан:
- остановить выполнение,
- изучить обязательные материалы,
- только после этого продолжить работу по задаче.
