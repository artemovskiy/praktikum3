## Инструкция по запуску МЛВ

Команда запуска (используется `clips` из `PATH`):

```bash
printf "(clear)\n(load \"./network_security_assessment.clp\")\n(reset)\n(watch rules)\n(run)\n(facts)\n(exit)\n" | clips
```

Пример реального запуска с полным выводом и пояснениями:
[notes/mlv_run_example.md](./mlv_run_example.md)

Расшифровка команды по частям:

- `printf "..."`:
  формирует текстовый сценарий команд для CLIPS.
- `\n`:
  перевод строки, разделитель между командами в сценарии.
- `(clear)`:
  очищает текущее состояние CLIPS, чтобы запуск начинался с чистого состояния.
- `(load "./network_security_assessment.clp")`:
  загружает файл с экспертной системой.
- `(reset)`:
  инициализирует систему начальными фактами из `deffacts`.
- `(watch rules)`:
  включает вывод срабатываний правил (строки `FIRE ...`).
- `(run)`:
  запускает механизм логического вывода.
- `(facts)`:
  печатает итоговый список фактов после выполнения правил.
- `(exit)`:
  завершает работу CLIPS.
- `| clips`:
  передает подготовленный сценарий на вход программе `clips`, чтобы команды выполнились автоматически.
